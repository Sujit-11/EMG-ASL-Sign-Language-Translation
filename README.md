# Sign Language Translation Based on Electromyography Using Deep Learning and Large Language Models

This repository contains the final Kaggle notebooks developed as part of the Master's thesis:

**Sign Language Translation Based on Electromyography Using Deep Learning and Large Language Models**

## Repository Contents

- **Stage1_Final.ipynb**
  - Data preprocessing
  - Signal normalization
  - Data augmentation
  - CNN–BiLSTM model training and evaluation
  - Letter classification

- **Stage2_Final.ipynb**
  - Word prediction using the trained Stage 1 model
  - Lexicon-based correction
  - LLM-based word correction
  - Performance evaluation

These notebooks represent the final implementation used to obtain the experimental results reported in the thesis.
